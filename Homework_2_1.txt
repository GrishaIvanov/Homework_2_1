В логической структуре БД для "Библиотеки" можно идентифицировать следующие сущности:

1. Сущность "Читатели":
   - Номер читательского билета (первичный ключ)
   - ФИО
   - Адрес
   - Телефон

2. Сущность "Книги":
   - Шифр книги (первичный ключ)
   - Название
   - Автор(ы)
   - Год издания
   - Объем (в страницах)
   - Цена
   - Количество экземпляров в фонде

3. Сущность "Издательства":
   - Название (первичный ключ)
   - Город

Связи между сущностями:

- Сущность "Книги" имеет связь с сущностью "Издательства". 

Одно издательство может быть связано с несколькими книгами, 
поэтому у сущности "Книги" будет внешний ключ,
указывающий на поле "Название" в сущности "Издательства".

- Сущность "Книги" также имеет связь с сущностью "Читатели".
Эта связь представляет, что каждая книга может быть взята несколькими читателями, 
и каждый читатель может иметь несколько книг. Для этой связи будет создана промежуточная таблица, 
которая будет содержать первичные ключи из сущностей "Книги" и "Читатели".

Выбор связей:
- Связь между сущностью "Книги" и "Издательства" устанавливается на основе того, 
что каждая книга должна быть связана с определенным издательством, и одно издательство может выпускать несколько книг.

- В случае связи между сущностью "Книги" и "Читатели", требуется промежуточная таблица для представления отношения "многие ко многим",
 поскольку каждая книга может быть взята несколькими читателями, и каждый читатель может брать несколько книг.

Первичные и внешние ключи:

- У сущности "Читатели" первичным ключом является поле "Номер читательского билета".
- У сущности "Книги" первичным ключом является поле "Шифр книги".
- У сущности "Издательства" первичным ключом является поле "Название".

- В сущности "Книги" будет внешний ключ "Название издательства", который связывается с полем "Название" в сущности "Издательства".

- Для связи между сущностями "Книги" и "Читатели" будет создана промежуточная таблица,
 содержащая внешние ключи "Шифр книги" и "Номер читательского билета", которые ссылается на первичные ключи соответствующих сущностей.
